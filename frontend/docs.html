MAE — Make Anything Editor

API documentation for the MAE editor backend. All endpoints are relative to http://127.0.0.1:8000.

Path System

%~/ — Project root (workspace directory). Example: %~/src/main.py → workspace/src/main.py
. prefix — Hidden files/folders. .git, .env etc. are NOT displayed in file tree or search results, but can still be read/written directly.
PrefixResolves toExample
%~/workspace/%~/main.py → workspace/main.py
%~/.configworkspace/.confighidden folder, not shown in tree

API Endpoints

AI

POST/api/ai/chat
Send a chat message. Supports Zen (free), Go (paid), and Ollama (local).
{
  "provider": "zen",
  "model": "deepseek-v4-flash-free",
  "messages": [{"role": "user", "content": "Hello"}],
  "stream": false,
  "system_prompt": "You are a helpful assistant"
}
GET/api/ai/models?provider=zen
List available models for a provider.
GET/api/ai/health
Check provider connection status.
{"zen": true, "go": false, "ollama": false}

Files

GET/api/files/tree?path=%~/&depth=5
Get file tree. Hidden files (.) excluded.
GET/api/files/list?path=%~/src
List directory contents.
GET/api/files/read?path=%~/main.py
Read file contents.
POST/api/files/write
Create or overwrite a file.
{"path": "%~/new.py", "content": "print('hello')"}
DELETE/api/files/delete?path=%~/old.py
Delete a file.
POST/api/files/mkdir?path=%~/new-folder
Create a directory.
POST/api/files/rename
Rename a file.
{"old_path": "%~/old.py", "new_name": "new.py"}
GET/api/files/search?q=query&path=%~/
Search files by name.

Execute

POST/api/execute/run
Execute code. Language: python or shell.
{"code": "print('hello')", "language": "python", "timeout": 30}

GitHub

GET/api/github/verify
Verify GitHub token. Header: X-GitHub-Token
GET/api/github/repos
List user repositories.
POST/api/github/create-repo
Create a new GitHub repository.
{"name": "my-project", "description": "Created with MAE", "private": false}
POST/api/github/push
Push files to a repository.
{"repo": "my-project", "files": [{"path":"main.py","content":"..."}], "message": "Update", "branch": "main"}

Extensions

GET/api/extensions/list
List installed extensions.
GET/api/extensions/get?name=python
Get extension details.
POST/api/extensions/install
Install an extension from manifest.
DELETE/api/extensions/uninstall?name=python
Uninstall an extension.
POST/api/extensions/validate
Validate an extension manifest.
